# image2pdf
## 概要
Python3.12と諸々のライブラリを用いて、画像ファイル群を1つのPDFファイルにします。  

## 動作確認環境
以下の環境にて動作確認済み。
|Windows       |MacOS (OSX)        |Linux|
|:------------:|:-----------------:|:---:|
|Windows10     |MacOS Catalina     |     |
|              |MacOS High Sierra  |     |　　

100枚程度のpng画像からの変換を動作確認として行いました。  


## その他  
メモリをかなり消費するので、フリーのメモリが4GB以上あるときに実行すると安定します。
